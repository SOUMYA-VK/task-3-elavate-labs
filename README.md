# task-3-elavate-labs
data visualization and powerpoint
1. Introduction


The purpose of this dashboard is to analyze and visualize car sales data across different years, makes, and sellers. Using Power BI, the dashboard provides key performance indicators, charts, and tables to help identify sales patterns, top-performing brands, and price trends.

 
2.Data Source
File name: task3.xlsx

Contents: Information about car make, model, year, seller, selling price, odometer, and other attributes.

Data was loaded into Power BI and cleaned for analysis.

3. Steps to Create the Dashboard
Import Data

Open Power BI Desktop and load the Excel file task3.xlsx.

Data Cleaning & Preparation

Verified column headers and removed empty rows.

Ensured numeric fields like sellingprice and odometer were set to the correct data type.

Checked for missing values and handled inconsistencies.

Creating Visuals

KPI Cards for total selling prices of Acura, Audi, and BMW.

Pie Chart for selling price by make.

Line Chart for count of models by year.

Bar Chart for selling price by seller.

Donut Chart for average selling price by year.

Table for selling price breakdown by year and make.

Adding Slicers

Added slicers for Year, Make, and Seller for interactive filtering.

Formatting

Applied consistent colors and labels.

Adjusted titles and legends for readability.

6. Conclusion
This Power BI dashboard provides a comprehensive overview of car sales data, enabling quick identification of top brands, sellers, and yearly trends. The interactive elements make it easy to explore the data and gain valuable insights for decision-making.
